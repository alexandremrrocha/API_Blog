# Blog do código

API para um blog, com autenticação, criptografia
e varias validações projeto em andamento...

## Stack utilizada

- Node
- Express
- Bcrypt
- JWT
- Passport
- SQLite

## Aprendizados

- Bcrypt
- JWT
- Passport
- SQLite

## Rodando localmente

- Clone o projeto

- Entre no diretório do projeto

- Instale as dependências

```bash
  npm install
```
- Inicie o servidor

```bash
  npm run dev
```
